阿里云服务器安装配置

Nov 09, 2021

双十一B站UP主给阿里云做活动，买云服务器满58返现60，本来想着白嫖一年的，但又想好好利用新用户的优惠，就买了三年的，170再返现60，相当于110买了三年，感觉很赚。。

服务器基本配置

阿里云ECS n4 1核 2G 1M带宽

操作系统：ubantu 20.04

安装openjdk1.8

  • 更新软件包列表
sudo apt-get update
  • 安装openjdk-8
sudo apt-get install openjdk-8-jdk
  • 查看java版本
java -version

  • 查看openjdk装在哪里
update-alternatives --config java
